Install

DATE: 2026 · OPERATOR: LOCAL · AUTH: GROQ_API_KEY REQUIRED

Get Daemon running locally

§ 1 · Clone and install
git clone https://github.com/Whauv/Daemon.git
cd Daemon
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
§ 2 · Configure your environment
copy .env.example .env
# add your GROQ_API_KEY in .env

Daemon currently requires a Groq API key for planning, execution assistance, and verification.

§ 3 · Launch the dashboard
python main.py --dashboard

The product opens in your browser and the dashboard is available locally.

END OF INSTALLATION GUIDE · Daemon v1.0
